CSS nesting & scoped styles

Utilisez l'intérupteur pour changer le CSS

Attention, ces fonctionalitées CSS ne fonctionnent en parallèles que sur les versions expérimentales de Google Chrome

La fonctionalitées CSS Nesting permet d'imbriqué des instructions CSS pour simplifier les écritures représentant les relations parent-enfant des éléments HTML d'une page, comme à la manière de SCSS.

La pseudo-classe CSS @scope permet de limiter les débordements de propriétées CSS sur des éléments parents ou voisin en limitant le rayon d'action de ces propriétées.
